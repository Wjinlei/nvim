### 介绍
这是我的vim配置, 它`fock`于`@theniceboy`.<br/>
它需要`neovim` >= `0.5`<br/>

![preview](./preview.jpg)

### 快速开始
```bash
mv ~/.config/nvim ~/.config/nvim.bak
git clone --depth 1 https://github.com/Wjinlei/nvim.git ~/.config/nvim
```
### 安装后,您需要
- [ ] 安装`pynvim`(pip)
- [ ] 安装`nodejs`
- [ ] 安装`neovim`(npm)
- [ ] 配置`python`路径, 编辑`_machine_specific.vim`
- [ ] 执行`:checkhealth`(检测有无错误)
